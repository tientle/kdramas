Search Dramas By: Title

  1. Put an underscore, _ ,for spaces and dashes in drama titles you want to search.
  2. Not case sensitive.
  3. Returns list of dramas containing the title/words inputed.
  4. Example: tientl.com/kdramas/title/vincenzo

Search Dramas By: Genre

  1. Put an underscore, _ ,for spaces and dashes in drama genres you want to search.
  2. Not case sensitive.
  3. Some genres include: Comedy, Romantic comedy, Crime, Fantasy, Business, Coming of age, Zombie, Family, Legal
  4. Example: tientl.com/kdramas/genre/comedy

Search Dramas By: Rating

  1. Put a whole number between 0 and 10.
  2. If you submit 7, it will return every drama rated from 7 to 7.9
  3. Example: tientl.com/kdramas/rating/7

Search Dramas By: Actor

  1. Put an underscore, _ ,for spaces and dashes in actor name.
  2. Not case sensitive
  3. Returns list of dramas containing the actor/words inputed. (ie. if you input the word "Kim", all dramas with actors with the last name Kim would appear)
  4. Example: Search for Kim Seon-Ho: tientl.com/kdramas/actor/kim_seon_ho

Search Dramas By: Year

  1. Input any year. Returns dramas that were released that year, if available.
  2. Example: tientl.com/kdramas/year/2020
